## Decisions（grill 2026-06-03 定案）

- **Keying**:measurement unit = **group**。D 側 query-dependent 表 + viz cache 全 key by `group_id`（錨點 question id / 成員 hash）。重型 `queries` registry → 砍成輕量 `submissions(id, query_text, group_id, submitted_at)`;strata / `in_final_set` 延後（報告挑選時補 `in_report` flag）。
- **冪等**:version-stamped(row 在 AND 版本==pin → skip);driver ①→⑦ 帶 `dirty` 旗,上游一動下游該 group 整段重跑(粗粒度)。
- **Providers**:① canonical **proxy**(`DuplicateChainGroups`,真 C 延後);⑤ 真 **PageRank authority**(yearly percentile;已完成 de-stage3 改名)。
- **Out-of-window**:`meta.db_window` + `n_out_of_window`(`source=="canonical"` / `year<2021`);per-point 不另加 flag。null authority 兩成因(窗外 / 在窗但作者不在 graph)別混。
- **Serve**:`GET /breakdown` = warm 讀 by `group_id`(miss→cold,不 assemble-on-miss);**`POST /queries` 第一線** = `submissions` + `BackgroundTasks(bake)` → 回 submission id → 輪詢;answer body **懶載** `GET /answer/{id}`(`/breakdown` 只回 stub)。
- **Names**:`source_name` `stage3_pagerank_pct` → `pagerank_yearly_pct`;`Stage3AuthorityProvider` → `PageRankAuthorityProvider`;DB `stage3_*` 表 + `Stage3Pipeline`/`Stage3Results`/`Stage3DB` 命名 migration 已完成(`user_authority` / `authority_run_meta` / `AuthorityPipeline` / `AuthorityStore` / `AuthorityDB`)。
- **import-linter**:6 規則寫進 ADR,enforcement 延後(spine 穩了再焊 CI)。
- **Scope**:單 query(q54069253 backfill)垂直打穿先 → query #2 才 live 全烤。
